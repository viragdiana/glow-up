# Glow Up

A full-stack self-knowledge dashboard for tracking personal growth, habits, and well-being over time.

## Overview

Glow Up is a personal analytics platform that helps you understand yourself better through structured self-reflection, habit tracking, and data-driven insights. Built with a React + TypeScript frontend and a .NET Web API backend, it is designed to be extensible — with AI-powered insights planned as a future layer.

## Planned Stack

| Layer | Technology |
|---|---|
| Frontend | React 18, TypeScript, Vite |
| Backend | .NET 9, ASP.NET Core Web API |
| Database | SQL Server (via Entity Framework Core) |
| Auth | ASP.NET Core Identity / JWT |
| AI (future) | Claude API (Anthropic) |
| Hosting | TBD |
